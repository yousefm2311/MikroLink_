# MikroLink

A public application project prepared for portfolio presentation with a clear user flow, reusable structure, and room for screenshots and technical documentation.

## Status

Public portfolio candidate.

## Features

- Application workflow implementation
- Responsive user-facing screens
- Reusable project structure
- Ready for screenshots, features, and case-study notes

## Tech Stack

- Flutter
- Dart

## Project Structure

```text
.
|-- lib/                # Flutter source code
|-- assets/             # Images, icons, fonts, and static assets
|-- test/               # Unit/widget tests when available
|-- pubspec.yaml        # Flutter dependencies and assets
|-- README.md           # Project documentation
```

## Getting Started

```bash
git clone https://github.com/yousefm2311/MikroLink_.git
cd MikroLink_
flutter pub get
flutter run
```

## Environment Variables

Create a local environment file only when the project requires external services.

```env
API_BASE_URL=
```

Never commit real secrets, API keys, or production credentials.
